# 🔢 Counter on Logic Elements

Цифровой счётчик, реализованный на **логических элементах и D-триггерах**.

Проект демонстрирует синтез цифрового автомата, реализующего заданную последовательность состояний.
Разработка включает получение логических функций, их минимизацию и построение схемы.

---

## ⚙️ Реализация

Счётчик построен на:

* D-триггерах
* комбинационной логике
* логических элементах **И / ИЛИ / НЕ**

Используется **4-разрядный счётчик**.

Последовательность состояний:

```
1011 → 0111 → 0100 → 0010 → 1111 → 0011 → 1000 → 0001 → 1100 → ...
```
